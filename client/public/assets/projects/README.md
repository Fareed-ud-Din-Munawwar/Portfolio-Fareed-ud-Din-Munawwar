# Project Images

Place your project images or GIFs in this folder.

## File Naming Convention

- `islam360.gif` - For Islam360 & Hindi Hadith 360 project
- `schoolgram.gif` - For Schoolgram project
- `alw.gif` - For ALW (Advanced Learning World) project
- `reel-stories.gif` - For Reel/Stories Application project

## Supported Formats

- GIF files (`.gif`)
- Image files (`.jpg`, `.jpeg`, `.png`, `.webp`)

Images will be automatically loaded from `/assets/projects/` path when referenced in the portfolio data.
